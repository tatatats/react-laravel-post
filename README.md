# React × Laravel 投稿アプリ

Reactを使って投稿画面を構築し、LaravelのAPIからデータの取得・登録・更新・削除を行うシンプルな投稿管理アプリです。

## 使用技術

- Laravel 11（REST API）
- React（Vite構成）
- React Router
- Axios
- Git / GitHub
